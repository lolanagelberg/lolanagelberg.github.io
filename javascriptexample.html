3. Getting User Input

Enter your name and get a personalized greeting:

Greeting will appear here!
JavaScript Code:
function greetUser() {
  const name = document.getElementById('name-input').value;
  const output = document.getElementById('greeting-output');
  output.innerHTML = `Hello, ${name}! Welcome to JavaScript!`;
  output.style.display = 'block';
}

4. Changing Element Styles

Click buttons to change the color of the box:

JavaScript Code:
function changeColor(color) {
  document.getElementById('color-box').style.backgroundColor = color;
}

5. Keeping Track of State

A simple counter that remembers its value:

Count: 0
JavaScript Code:
let counter = 0;
function incrementCounter() {
  counter++;
  updateCounterDisplay();
}
function updateCounterDisplay() {
  document.getElementById('counter-display').textContent = `Count: ${counter}`;
}
